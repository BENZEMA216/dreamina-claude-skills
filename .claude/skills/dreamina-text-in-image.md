# 图片文字规范

## 引号规则
- **必须使用中文双引号 ""** 包裹文字内容
- **禁止**使用英文双引号 " 或单引号 '（会导致文字效果劣化）

## 必须明确的属性
- 字体风格
- 颜色
- 在画面中的位置

## 示例
正确：`画面左上角有手写体风格的"你好"和"Hello"，颜色为黑色`

错误：`画面左上角有"Hello"`（缺少字体、颜色、位置）

## 表情包特殊规则
- 必须加入搞笑文字
- 文字是表情包的灵魂，必须成为核心表达点
