# 参考词表创建指南

## 角色参考词表

为每个角色定义固定的外貌描述词，确保全篇一致。

### 必含字段

| 字段 | 说明 | 示例 |
|------|------|------|
| 角色ID | 唯一标识 | CHAR_01 |
| 角色名 | 角色姓名 | 齐夏 |
| 年龄 | 具体年龄 | 25岁 |
| 发型/发色 | 具体描述 | 短黑发 |
| 面部特征 | 眼睛、脸型等 | 锐利的分析型眼神 |
| 服装 | 具体服装描述 | 脏污的灰色T恤、深色牛仔裤 |
| 体型 | 身材描述 | 中等身材 |
| 标志性特征 | 纹身、眼镜等 | 无 |

### 示例

```markdown
| 角色ID | 角色名 | 固定描述词 |
|--------|--------|-----------|
| CHAR_01 | 齐夏 | 25岁中国男性，短黑发，锐利的分析型眼神，穿着脏污的灰色T恤和深色牛仔裤 |
| CHAR_02 | 山羊头 | 高瘦男性，戴着腐朽的山羊头骨面具，发黄打结的毛发，黑色正装西服，苍白细长的双手 |
```

### 服装状态变化

角色服装可能随剧情变化，需要标记：

| 状态 | 变化 |
|------|------|
| 初始 | 脏污、破损程度 |
| 事件后 | 血迹、撕裂、丢失物品 |

## 场景参考词表

为每个场景/场景状态定义固定的环境描述词。

### 必含字段

| 字段 | 说明 |
|------|------|
| 场景ID | 唯一标识 |
| 场景名 | 场景描述名 |
| 环境描述 | 空间、物品、布局 |
| 光照条件 | 光源、亮度、色温 |
| 氛围 | 情绪、风格 |

### 场景状态区分

同一场景在不同剧情阶段可能有不同状态：

```markdown
| 场景ID | 场景名 | 固定描述词 |
|--------|--------|-----------|
| SCENE_A | 主场景-杀人前 | 密封混凝土房间，墙面地板天花板有网格纹理，无门无窗，老旧钨丝灯悬挂中央投射昏黄灯光，斑驳木质圆桌居中，繁复花纹座钟，10把木椅环绕，幽闭恐怖氛围 |
| SCENE_B | 主场景-杀人后 | [SCENE_A] + 桌面血迹和脑浆飞溅，无头尸体瘫坐椅上，幸存者脸上沾血 |
```

### 回忆/闪回场景

回忆场景需要视觉区分：

```markdown
| 场景ID | 场景名 | 固定描述词 |
|--------|--------|-----------|
| SCENE_C | 回忆-陕西小巷 | 夜晚中国城市狭窄小巷，停放的轿车，头顶大型广告牌，地震晃动，灰尘飘落，暖橙色路灯光，**闪回柔和晕边效果** |
```

关键：回忆场景必须添加 **闪回柔和晕边效果** 与现实场景区分。
