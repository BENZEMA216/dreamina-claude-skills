# 分镜表检查流程

## 三遍检查法

### 第一遍：逐镜头检查

逐个检查每个分镜：

| 检查项 | 说明 |
|--------|------|
| 景别匹配 | 景别是否与可见内容匹配？远景是否包含所有应有元素？ |
| 场景状态 | 场景状态是否正确？是否反映了之前的事件影响？ |
| 角色描述 | 角色描述是否使用参考词表？是否统一？ |
| 提示词完整 | 提示词是否包含所有必要元素？ |
| 语言规范 | 提示词是否使用中文？ |

### 第二遍：连续性检查

检查相邻分镜之间的连贯性：

| 检查项 | 说明 |
|--------|------|
| 元素连贯 | 相邻分镜是否逻辑连贯？ |
| 无凭空出现 | 是否有元素"凭空出现"或"凭空消失"？ |
| 状态转折 | 状态变化是否有明确转折点？ |
| 动作连贯 | 角色动作是否有合理的前后关系？ |

### 第三遍：一致性检查

全局检查统一性：

| 检查项 | 说明 |
|--------|------|
| 角色一致 | 同一角色的描述是否全篇统一？ |
| 场景一致 | 同一场景的描述是否全篇统一？ |
| 光照一致 | 光照条件是否在同一场景内统一？ |
| 色调一致 | 色调是否与场景状态匹配？ |

---

## 常见错误清单

| 错误类型 | 描述 | 解决方案 |
|----------|------|----------|
| **景别逻辑错误** | 远景/全景中遗漏应有元素 | 检查景别与可见性是否匹配 |
| **状态遗漏** | 事件发生后未更新场景状态 | 标记状态变化点，逐一检查后续分镜 |
| **角色描述不一致** | 同一角色在不同分镜描述不同 | 使用角色参考词表 |
| **光照不一致** | 同一场景光照条件变化 | 统一光照描述词 |
| **凭空出现** | 元素没有铺垫就突然出现 | 检查镜头连续性 |
| **回忆无区分** | 回忆与现实视觉风格相同 | 添加闪回效果（柔和晕边） |
| **服装矛盾** | 角色服装与参考词表不符 | 核对角色参考词表 |
| **提示词使用英文** | 未按规范使用中文 | 改为中文提示词 |

## 输出检查清单模板

在分镜表末尾添加检查清单：

```markdown
## 一致性检查清单

### 角色一致性 ✓
- [x] 齐夏: 所有分镜使用统一外貌描述
- [x] 山羊头: 所有分镜使用统一描述
- [ ] ...

### 场景一致性 ✓
- [x] 状态变化后所有分镜包含正确状态描述
- [x] 所有分镜包含统一光照描述
- [x] 回忆场景使用闪回效果区分
- [ ] ...

### 技术参数一致性 ✓
- [x] 所有分镜统一使用指定画幅
- [x] 所有分镜使用中文提示词
- [ ] ...
```
