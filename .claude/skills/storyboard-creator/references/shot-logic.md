# 镜头逻辑原则

## 景别与可见性原则

**核心规则：景别决定了画面中必然可见的元素**

| 景别 | 可见范围 | 创作要点 |
|------|----------|----------|
| **远景/全景** | 整个场景 | 场景中的所有主要元素（人物、道具、环境）都应该可见，不能"隐藏"本应出现的元素 |
| **中景** | 人物膝盖以上或环境局部 | 如果拍摄包含人物的区域，人物必须出现 |
| **近景** | 人物胸部以上 | 聚焦单个或少数人物 |
| **特写** | 局部细节 | 可以排除其他元素，只聚焦特定物体/表情 |

### 错误示例

```
❌ 场景号 1-1-02
   镜头类型：远景
   描述：展现封闭房间全貌
   问题：远景必然看到房间里的所有人，prompt中却没有人物
```

### 正确做法

```
✓ 如果想制造悬念，用特写/局部镜头避开人物区域
✓ 如果用远景/全景，必须包含场景中应有的所有元素
```

---

## 镜头连续性原则

**核心规则：相邻分镜的画面内容必须逻辑连贯**

- 不能前一个镜头没有人，后一个镜头突然出现人
- 物品位置、人物状态必须保持连续
- 场景状态变化（如血迹出现）必须有明确的转折点

### 错误示例

```
❌ 1-1-02: 空房间全景
   1-1-05: 十人围坐
   问题：人是从哪里"冒出来"的？
```

### 正确做法

```
✓ 1-1-02: 房间全景（包含沉睡人物剪影）
   1-1-03: 中景圆桌+沉睡众人
   → 人物从一开始就存在，逐步清晰
```

## 悬念构建技巧

如果想要制造"揭晓"效果，应该使用以下技巧：

| 技巧 | 说明 | 示例 |
|------|------|------|
| **特写规避** | 用特写镜头只拍局部，避开想隐藏的元素 | 只拍桌面中央的座钟，不拍桌边 |
| **俯拍限制视野** | 垂直俯拍只看到桌面，看不到椅子区域 | 正上方俯拍圆桌 |
| **前景遮挡** | 用前景物体遮挡部分画面 | 灯泡占据画面主体，背景虚化 |
| **光影隐藏** | 利用黑暗/阴影隐藏部分内容 | 人物在黑暗中只有剪影 |
| **影子铺垫** | 先展示影子，再揭示本体 | 山羊头的影子投射在桌面上 |

### 悬念序列示例

```
1-1-01: 灯特写（建立氛围）
1-1-02: 房间全景 + 沉睡人物剪影（首次展示）
1-1-03: 中景圆桌 + 沉睡众人（更清晰）
1-1-04: 座钟特写（时间线索）
1-1-05: 俯拍全景 + 山羊头影子（铺垫）
1-1-06: 山羊头出场（揭晓）
```
