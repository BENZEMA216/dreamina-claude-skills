# 张力曲线分析指南

## 什么是张力曲线

张力曲线是衡量观众情绪投入程度随时间变化的可视化工具。一个好的分镜应该呈现"波浪式上升"的张力曲线，而非平铺直叙。

```
理想张力曲线：

张力
  ↑
5 │                         ████ 高潮
  │                      ███░░░░███
4 │                   ███░░░░░░░░░░███ 余震
  │            ███████░蓄势░░░░░░░░░░░
3 │      ██████░░░░░░░░░░░░░░░░░░░░░░░
  │  ████░░小起伏░░░░░░░░░░░░░░░░░░░░░
2 │██░铺垫░░░░░░░░░░░░░░░░░░░░░░░░░░░
  │░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░

  └──────────────────────────────────→ 时间
```

## 叙事节拍类型

| 节拍 | 张力等级 | 功能 | 典型时长占比 |
|------|----------|------|--------------|
| **开场·建立** | ★☆☆☆☆ | 空间介绍、氛围铺垫、悬念植入 | 15-20% |
| **发展·铺垫** | ★★☆☆☆ | 信息交换、角色展示、冲突酝酿 | 30-40% |
| **蓄势** | ★★★★☆ | 高潮前的悬念积累、不祥预感 | 10-15% |
| **高潮** | ★★★★★ | 核心事件爆发、视觉震撼 | 5-10% |
| **余震** | ★★★★☆ | 反应呈现、心理冲击、权力确立 | 10-15% |
| **沉淀** | ★★☆☆☆ | 情绪沉淀、悬念留白 | 5-10% |

## 常见张力问题

### 1. 平铺直叙

```
问题曲线：
张力 ─────────────────────────────
                 (缺乏起伏)
```

**症状**：
- 整段分镜张力维持在2-3星
- 没有明显的高潮点
- 观众难以感受到情绪变化

**解决方案**：
- 在铺垫段落增加"小高潮"
- 明确设计蓄势-高潮-余震结构
- 增加对比蒙太奇创造张力跳跃

### 2. 高潮过短

```
问题曲线：
张力          █ ← 仅1-2秒
         ████░██
     ████░░░░░░░░███
```

**症状**：
- 核心事件仅用1-2个镜头表现
- 视觉冲击被压缩
- 观众来不及"处理"情绪

**解决方案**：
- 将单一高潮镜头拆分为多个快切
- 使用慢镜头延长关键瞬间
- 增加反应镜头扩展高潮段落

### 3. 缺乏呼吸

```
问题曲线：
张力 ███████████████████
         (持续高压)
```

**症状**：
- 连续的信息密集镜头
- 没有空镜或静物镜头
- 对话段落没有间隙

**解决方案**：
- 在高张力段落后插入"喘息"镜头
- 使用静物/环境镜头调节节奏
- 延长某些镜头时长提供处理时间

### 4. 铺垫过长

```
问题曲线：
张力                    ████
     ████████████████████░░░
         (铺垫占比过高)
```

**症状**：
- 铺垫段落超过50%的篇幅
- 观众耐心流失
- 高潮显得突兀

**解决方案**：
- 在铺垫中插入悬念点
- 删减非必要的信息镜头
- 合并功能重复的镜头

## 张力曲线绘制方法

1. **列出所有分镜**，标注场景号
2. **为每个分镜评分**（1-5星）
3. **识别节拍边界**（开场/铺垫/蓄势/高潮/余震）
4. **绘制曲线图**
5. **诊断问题区域**

### 示例

```markdown
| 分镜范围 | 节拍 | 平均张力 | 问题诊断 |
|----------|------|----------|----------|
| 01-07 | 开场 | ★★☆☆☆ | ✓ 正常 |
| 08-19 | 铺垫 | ★★★☆☆ | ⚠️ 过长，缺乏起伏 |
| 20-21 | 蓄势 | ★★★★☆ | ✓ 有效 |
| 22 | 高潮 | ★★★★★ | ⚠️ 过短 |
| 23-25 | 余震 | ★★★★☆ | ✓ 正常 |
```
