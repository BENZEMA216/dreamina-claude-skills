# 分镜优化模式库

## 构图优化

### 问题：中央构图过多

**诊断标准**：中央构图占比 > 50%

**优化策略**：

| 原构图 | 替换方案 | 适用场景 |
|--------|----------|----------|
| 中央构图（对话） | 越肩构图 | 两人对话 |
| 中央构图（单人） | 三分法（左/右） | 角色独白/思考 |
| 中央构图（群像） | 对角线构图 | 冲突/紧张 |
| 中央构图（物件） | 三分法 | 叙事道具 |

**保留中央构图的场景**：
- 仪式感开场（灯泡、座钟）
- 对称场景（俯拍圆桌）
- 高潮冲击（暴力特写）
- 权力宣示（反派正面）

---

### 问题：景别单调

**诊断标准**：连续3个以上相同景别

**优化策略**：

```
错误：近景→近景→近景→近景
正确：近景→中景→近景→特写
```

**景别节奏公式**：
- 开场：特写→远景→中景（收→放→稳）
- 对话：中景→近景→近景→中景（建立→聚焦→聚焦→呼吸）
- 高潮：中景→近景→特写→全景（靠近→冲击→余震）

## 节奏优化

### 问题：对话段落过平

**诊断标准**：连续5个以上"单人近景+台词"

**优化策略**：

1. **插入反应镜头**
   ```
   A说话 → B反应 → A继续 → C反应
   ```

2. **插入空镜**
   ```
   对话1 → 对话2 → 座钟特写 → 对话3
   ```

3. **使用双人/群像镜头**
   ```
   A单人 → A+B双人 → B单人
   ```

4. **增加运镜变化**
   ```
   静止 → 静止 → 缓推 → 静止
   ```

---

### 问题：高潮过短

**诊断标准**：核心事件仅1-2个镜头，总时长<3秒

**优化策略**：

**拆分法** — 将单一高潮镜头拆分为多个快切：

```
原版：
1-1-22 | 暴力撞击 | 2秒

优化版：
1-1-27 | 撞击瞬间 | 0.5秒（动态模糊）
1-1-28 | 血液飞溅 | 1.5秒（慢镜头）
1-1-29 | 脑组织落下 | 1秒（细节）
```

**扩展法** — 增加前后反应镜头：

```
原版：
蓄势 → 高潮 → 余震

优化版：
蓄势 → 高潮1 → 高潮2 → 高潮3 → 反应1 → 反应2 → 余震
```

### 问题：缺乏呼吸

**诊断标准**：连续10个以上信息密集镜头，无空镜

**优化策略**：

**呼吸镜头类型**：

| 类型 | 示例 | 功能 |
|------|------|------|
| 静物特写 | 座钟、灯泡 | 时间流逝感 |
| 环境空镜 | 墙壁网格 | 空间压迫感 |
| 反应镜头 | 无台词的角色表情 | 情绪沉淀 |
| 细节特写 | 手指颤抖 | 心理暗示 |

**插入位置**：
- 对话段落每5-7个镜头插入1个呼吸镜头
- 高张力段落后必须有呼吸镜头
- 节拍转换点前后

## 空间优化

### 问题：空间关系模糊

**诊断标准**：角色位置关系不清，观众困惑"谁在哪"

**优化策略**：

**增加确立性镜头**：

```
问题序列：
山羊头特写 → 众人苏醒
（山羊头在哪？与众人什么关系？）

优化序列：
山羊头特写 → 【新增：山羊头站在众人身后全景】 → 众人苏醒
```

**使用视线匹配**：

```
问题序列：
齐夏观察 → 封闭墙壁

优化序列：
齐夏观察（眼神方向明确） → 齐夏视角POV（看到封闭墙壁）
```

## 运镜优化

### 问题：运镜利用不足

**诊断标准**：>80%镜头为"静止"

**优化策略**：

**有叙事功能的运镜**：

| 运镜 | 心理效果 | 适用场景 |
|------|----------|----------|
| 缓推 | 靠近、压迫、聚焦 | 威胁靠近、重要细节 |
| 缓拉 | 远离、揭示、孤独 | 揭示全貌、角色孤立 |
| 左/右摇 | 扫视、寻找、展示 | 环境介绍、群像 |
| 下移 | 揭示、发现 | 从上往下揭示空间 |
| 跟拍 | 跟随、参与 | 角色移动 |
| 焦点变化 | 注意力转移 | 前后景关系 |

**运镜分布建议**：
- 静止镜头：60-70%
- 运动镜头：30-40%

## Prompt优化

### 问题：英文prompt

**优化策略**：全部改为中文

### 问题：格式不统一

**优化策略**：统一格式

```
中文Prompt标准格式：
[景别][运镜]，[主体描述]，[动作/表情]，[环境描述]，[光照条件]，[氛围关键词]，[技术参数]

示例：
中景镜头缓推，25岁中国男性短黑发锐利分析型眼神穿脏污灰色T恤，
手抚下巴沉思表情，密封混凝土房间网格纹理背景，
昏黄钨丝灯光，悬念氛围，电影感，16:9画幅
```
