## 快速检查清单

### 张力曲线检查 ✓

- [ ] 是否绘制了张力曲线图？
- [ ] 是否划分了叙事节拍（开场/铺垫/蓄势/高潮/余震）？
- [ ] 是否识别出张力问题（过平/高潮过短/缺乏呼吸/铺垫过长）？

### 逐镜头批注检查 ✓

- [ ] 是否完成所有镜头的四维度分析？
- [ ] 是否为每个镜头给出张力评分（1-5星）？
- [ ] 是否标注了亮点镜头（✓✓）和问题镜头（⚠️）？
- [ ] 是否给出了具体的改进建议？

### 问题汇总检查 ✓

- [ ] 是否汇总了所有优点？
- [ ] 是否汇总了所有问题及解决方案？
- [ ] 是否列出了需新增的镜头及位置？

### 优化版生成检查 ✓

- [ ] 是否应用了所有改进建议？
- [ ] Prompt是否改为中文？
- [ ] 是否生成了优化对照表？
- [ ] 是否更新了一致性检查清单？
- [ ] 构图分布是否优化（中央构图<50%）？

## 四维度速查

| 维度 | 核心问题 | 符号 |
|------|----------|------|
| 🎬 叙事功能 | 这个镜头为什么存在？ | ✓/✓✓/⚠️ |
| 🎨 美学评价 | 这个镜头好看吗？ | ✓/⚠️ |
| ⚡ 张力定位 | 这个镜头的情绪强度？ | ★☆☆☆☆~★★★★★ |
| 🔗 剪辑逻辑 | 前后镜头连贯吗？ | ✓/⚠️ |

---

## 常见问题-解决方案速查

| 问题 | 快速诊断 | 解决方案 |
|------|----------|----------|
| 对话过平 | 连续5+单人近景 | 插入反应镜头、越肩、空镜 |
| 高潮过短 | 核心事件<3秒 | 拆分为3个快切 |
| 缺乏呼吸 | 连续10+信息镜头 | 每5-7镜插入1个呼吸镜头 |
| 构图单调 | 中央构图>50% | 增加三分法、越肩、对角线 |
| 空间模糊 | 角色位置不清 | 增加确立性镜头 |
| 运镜不足 | 静止镜头>80% | 增加缓推/摇/焦点变化 |
