# 逐镜头批注指南

## 批注模板

```markdown
### [场景号] | [镜头描述简称]

**基本信息**：[镜头类型] | [构图] | [色调] | [运镜] | [时长]

**视听语言分析**：

- 🎬 **叙事功能**：
  - [这个镜头在叙事中的作用]
  - [信息揭示的顺序和节奏]
  - [视点控制分析]

- 🎨 **美学评价**：
  - [构图法则运用]
  - [色彩与情绪匹配]
  - [光影戏剧性]

- ⚡ **张力定位**：[★评分] —— [在情绪曲线中的位置]

- 🔗 **剪辑逻辑**：
  - [与前后镜头的关系]
  - [衔接是否流畅]

**改进建议**：
1. [建议1]
2. [建议2]

**张力评分**：★★★☆☆
```

---

## 四维度详解

### 维度一：叙事功能

**评估要点**：

| 评估项 | 问题 | 示例 |
|--------|------|------|
| **景别叙事功能** | 为什么用这个景别？ | 特写聚焦情绪，远景建立空间 |
| **视点控制** | 谁在看？ | 主观镜头=角色视角，客观镜头=上帝视角 |
| **信息层次** | 揭示了什么信息？ | 先整体后细节，还是先细节后整体 |
| **留白与暗示** | 没有直接展示什么？ | 影子暗示存在，声音暗示画外 |
| **运镜意义** | 运镜传达什么心理？ | 推=靠近/压迫，拉=远离/揭示 |

**常用标注**：
- ✓ 叙事功能明确
- ✓✓ 叙事功能出色（亮点镜头）
- ⚠️ 叙事功能模糊
- ❌ 叙事功能缺失

---

### 维度二：视觉美学

**构图法则检查**：

| 构图类型 | 适用场景 | 心理效果 |
|----------|----------|----------|
| **中央构图** | 仪式感、对称场景 | 稳定、庄严 |
| **三分法** | 大多数场景 | 平衡、自然 |
| **对角线** | 动态、冲突 | 紧张、不稳定 |
| **越肩镜头** | 对话、对峙 | 参与感、对抗 |
| **框架构图** | 窥视、隔离 | 疏离、监视 |

**色彩情绪检查**：

| 色调 | 情绪 | 适用场景 |
|------|------|----------|
| 暖黄 | 温暖、希望、怀旧 | 光源、回忆 |
| 冷青 | 压抑、恐惧、疏离 | 恐怖、悬疑 |
| 血红 | 暴力、危险、愤怒 | 血腥、冲突 |
| 金色 | 神圣、珍贵、回忆 | 闪回、高光时刻 |

**光影检查**：
- 顺光：平淡、纪实
- 侧光：立体、戏剧性
- 逆光：剪影、神秘
- 底光：恐怖、不安

---

### 维度三：张力定位

**评分标准**：

| 星级 | 情绪曲线位置 | 观众状态 |
|------|--------------|----------|
| ★☆☆☆☆ | 开场/建立 | 观察、进入 |
| ★★☆☆☆ | 铺垫/信息 | 接收、理解 |
| ★★★☆☆ | 发展/悬念 | 好奇、期待 |
| ★★★★☆ | 蓄势/威胁 | 紧张、担忧 |
| ★★★★★ | 高潮/爆发 | 震撼、恐惧 |

**标注格式**：
```
⚡ **张力定位**：★★★★☆ —— 蓄势阶段，死亡预兆
```

---

### 维度四：剪辑逻辑

**衔接检查**：

| 检查项 | 说明 |
|--------|------|
| **景别递进** | 远景→中景→近景→特写是否合理？ |
| **视线匹配** | "看"镜头后是否跟着"所见"镜头？ |
| **动作连贯** | 动作是否跨镜头连续？ |
| **轴线规则** | 180度轴线是否遵守或有意打破？ |
| **时空连续** | 时间和空间是否连贯？ |

**常见问题标注**：
- ⚠️ 跳跃过大
- ⚠️ 视线不匹配
- ⚠️ 轴线跳跃
- ⚠️ 空间关系模糊

## 批注示例

### 1-1-05 | 俯拍全景+山羊头影子

**基本信息**：全景（俯拍） | 对称构图 | 冷青色 | 静止 | 4秒

**视听语言分析**：

- 🎬 **叙事功能**：
  - ✓✓ **亮点镜头**：影子的出现是绝佳的悬念设计
  - ✓ 俯拍视角暗示"被监视"、"猎物"的处境
  - ✓ 在正式揭示山羊头前制造"不祥预感"

- 🎨 **美学评价**：
  - ✓ 俯拍的上帝视角创造了"命运棋盘"的隐喻
  - ✓ 影子斜跨桌面的构图打破了对称，制造不安
  - ✓ 冷青色强化了"死亡阴影"的感觉

- ⚡ **张力定位**：★★★☆☆ —— 悬念升级，威胁显现

- 🔗 **剪辑逻辑**：
  - ✓ 从04的物件特写到05的空间俯瞰，视角跳跃创造了"旁观者"的恐惧
  - ✓ 为06的山羊头正式出场做了完美铺垫

**改进建议**：
1. 这是目前最有创意的镜头之一，建议保留
2. 可考虑影子"缓慢延伸"的动态描述，增加威胁感

**张力评分**：★★★★☆

## 批注效率技巧

1. **先快速扫描**：标记明显的问题镜头和亮点镜头
2. **聚焦问题区域**：对话段落、高潮段落重点批注
3. **使用符号系统**：✓/✓✓/⚠️/❌ 快速标注
4. **批量处理相似问题**：如"连续中央构图"可一起指出
5. **改进建议要具体**：避免"可以更好"这类模糊建议
