# 可灵（Kling）Prompt 生成规则

为千川素材生成可灵 AI 的图片和视频 Prompt。可灵使用中文 prompt。

## 一、可灵图片 Prompt

### Prompt 结构
```
[主体描述]，[场景环境]，[光影氛围]，[色调风格]，[构图方式]
```

### 要求
- 使用中文描述
- 主体明确，描述具体
- 标注参考风格：写实 / 插画 / 3D / 动漫
- 画面比例默认 9:16（竖屏）

### 输出模板
```
--- 镜号 X：[画面描述摘要] ---

【可灵图片 Prompt】
[中文 prompt]
参考风格：写实
画面比例：9:16
```

### 常用模式

#### 产品特写
```
[产品名称]放置在[表面材质]上，[产品细节描述]，柔和的影棚灯光，商业产品摄影风格，干净背景，特写镜头
```

#### 使用场景
```
一位[年龄][性别]在[场景]中使用[产品]，[表情/动作]，自然光线，生活方式摄影，温暖色调，中景
```

## 二、可灵视频 Prompt

### Prompt 结构
```
首帧描述：[起始画面的详细描述]
尾帧描述：[结束画面的详细描述]（可选，用于精确控制运动）
运动描述：[主体运动 + 镜头运动]
```

### 参数说明

| 参数 | 说明 | 推荐值 |
|------|------|--------|
| 时长 | 生成视频时长 | 5s（标准）/ 10s（需拼接） |
| 模式 | 生成质量 | 标准（快速迭代）/ 专业（最终输出） |
| 首帧图 | 可上传起始帧图片 | 建议先用图片 prompt 生成首帧再图生视频 |
| 尾帧图 | 可上传结束帧图片 | 用于精确控制运动方向 |

### 运动描述关键词

#### 镜头运动
- 推进/拉远：镜头缓慢推进 / 镜头逐渐拉远
- 平移：镜头从左向右平移 / 镜头从右向左平移
- 环绕：镜头围绕主体缓慢旋转
- 俯仰：镜头从上往下 / 镜头从下往上
- 固定：镜头固定不动

#### 主体运动
- 人物：缓慢转头、拿起产品、微笑、走动
- 产品：缓慢旋转、光影流动、水珠滑落
- 环境：风吹动窗帘、光影变化、水面波动

### 输出模板
```
--- 镜号 X：[画面描述摘要] ---

【可灵 AI 视频】
首帧描述：[详细中文描述]
尾帧描述：[详细中文描述]
运动描述：[镜头运动 + 主体运动]
时长：5s
模式：专业

【首帧图建议】：建议先用可灵图片生成首帧，再用图生视频获得更好效果
```

### 千川素材常用视频 Prompt

#### 产品旋转展示
```
首帧描述：[产品]正面特写，放置在[表面]上，柔和灯光，干净背景
运动描述：产品缓慢顺时针旋转，镜头固定，光影随旋转自然变化
时长：5s
模式：专业
```

#### 场景使用演示
```
首帧描述：一位[人物描述]手持[产品]，站在[场景]中，自然光线
尾帧描述：[人物]将[产品]展示向镜头，微笑，产品占据画面中心
运动描述：镜头缓慢推进，人物将产品从身侧举到面前
时长：5s
模式：专业
```

#### 效果展示
```
首帧描述：[使用前状态的详细描述]
尾帧描述：[使用后效果的详细描述]
运动描述：画面从左向右缓慢过渡，呈现前后对比效果
时长：5s
模式：专业
```
