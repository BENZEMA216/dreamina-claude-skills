# 后期合成方案

千川素材的后期合成规范，包括剪辑、合成、输出的完整流程。

## 推荐工具

| 工具 | 适用场景 | 优势 |
|------|----------|------|
| 剪映专业版 | 国内投放首选 | 与抖音生态打通，模板丰富 |
| CapCut | 海外投放 | 剪映海外版，功能相同 |
| Premiere Pro | 专业制作 | 功能最全面 |
| DaVinci Resolve | 调色需求高 | 专业调色工具 |
| Final Cut Pro | Mac 用户 | 性能优秀 |

## 剪辑工作流程

### 1. 素材整理
```
项目文件夹结构：
/项目名/
├── 01_原始素材/
│   ├── AI生成/
│   ├── 实拍/
│   └── 产品图/
├── 02_音频/
│   ├── 配音/
│   └── BGM/
├── 03_工程文件/
└── 04_输出/
```

### 2. 时间线搭建
1. 先铺配音音轨，确定整体节奏
2. 根据配音节奏放置视频素材
3. 添加 BGM，调整音量平衡
4. 添加字幕
5. 添加转场和特效
6. 调色和最终检查

### 3. 音频混合
```
音量层级建议：
- 配音：0dB（基准）
- BGM：-15dB 到 -20dB（配音的 20-30%）
- 音效：-10dB 到 -15dB（按需）

处理建议：
- 配音做降噪处理
- BGM 在配音处自动降低（剪映的"音量跟随"）
- 结尾 BGM 渐弱
```

## 转场方式

### 推荐转场
| 转场 | 适用场景 | 剪映设置 |
|------|----------|----------|
| 硬切 | 快节奏、卖货感 | 无转场 |
| 叠化 | 平稳过渡 | 交叉溶解 0.3-0.5s |
| 闪白/闪黑 | 强调切换 | 闪白/闪黑 0.1-0.2s |
| 缩放 | 场景切换 | 缩放转场 |
| 滑动 | 对比、多产品 | 左右/上下滑动 |

### 避免的转场
- 3D 翻转、卷页等过于花哨的效果
- 持续时间超过 0.5 秒的转场
- 与内容风格不搭的转场

## 特效建议

### 常用特效
| 特效 | 用途 | 注意事项 |
|------|------|----------|
| 放大强调 | 突出产品细节 | 1.2-1.5x，不要太夸张 |
| 速度变化 | 节奏调节 | 慢动作突出质感 |
| 模糊背景 | 突出主体 | 高斯模糊 10-20% |
| 色彩增强 | 提升质感 | 适度，不要失真 |
| 光效 | 增加氛围 | 克制使用 |

### 避免的特效
- 过多的 emoji 和贴纸
- 抖动、故障等干扰阅读的效果
- 与产品调性不符的特效

## 调色建议

### 千川素材调色原则
- 明亮、通透：提升画面清晰度
- 肤色自然：人物肤色不要偏色
- 产品真实：产品颜色不要失真
- 统一风格：全片色调保持一致

### 基础调色参数参考
```
曝光：+5 到 +15（提亮）
对比度：+10 到 +20
饱和度：+5 到 +15（轻微提升）
色温：根据场景微调
清晰度：+5 到 +10
```

## 输出规格

### 抖音千川标准规格
```
分辨率：1080 x 1920（9:16 竖屏）
帧率：30fps（标准）/ 60fps（高质量）
码率：8-15 Mbps
编码：H.264
音频：AAC, 48kHz, 立体声
格式：MP4
```

### 其他平台适配
| 平台 | 分辨率 | 比例 |
|------|--------|------|
| 抖音/千川 | 1080x1920 | 9:16 |
| 快手 | 1080x1920 | 9:16 |
| 小红书 | 1080x1440 | 3:4 |
| 微信视频号 | 1080x1920 | 9:16 |
| B站 | 1920x1080 | 16:9 |
| YouTube Shorts | 1080x1920 | 9:16 |

## 质检清单

在输出前逐项检查：
- [ ] 视频时长符合要求（7-60秒）
- [ ] 画面无黑边、无变形
- [ ] 配音与画面同步
- [ ] 字幕无错别字，时机正确
- [ ] BGM 不盖过配音
- [ ] 结尾行动引导清晰
- [ ] 无版权风险素材
- [ ] 手机预览效果正常
