# 七种脚本类型 & AIGC 适配度

根据产品特性、客单价和目标选择最合适的脚本类型。

## 脚本类型速查表

| 类型 | 时长 | 适用场景 | 客单价 | AIGC 适配度 |
|------|------|----------|--------|-------------|
| 情景种草 | 20-40s | 多卖点产品 | 中低 | ★★★★★ |
| 素人口播 | 15-30s | 大众消费品 | 低 | ★★★★☆ |
| 达人测评 | 30-60s | 中高客单产品 | 中高 | ★★★☆☆ |
| 产品展款 | 7-15s | 视觉冲击强的产品 | 不限 | ★★★★★ |
| 剧情演绎 | 30-60s | 品宣为主 | 中高 | ★★★★☆ |
| 卖家促销 | 15-30s | 高性价比产品 | 低 | ★★★★☆ |
| 明星测评 | 30-60s | 品牌产品 | 高 | ★★☆☆☆ |

## 各类型详解

### 情景种草（AIGC 首选）
- **核心**：从买家视角展示使用场景
- **结构**：痛点引入 → 产品出场 → 使用效果 → 行动引导
- **AIGC 方式**：AI 生成生活场景 + 产品实拍素材合成
- **适合**：护肤、家居、食品、日用品等多卖点品类

### 素人口播
- **核心**：真人安利，对出镜人物亲和力要求高
- **AIGC 方式**：数字人 + AI 配音，或真人拍摄 + AI 后期增强
- **注意**：数字人质量直接影响信任感，建议用高质量数字人方案

### 达人测评
- **核心**：强达人背书，增加信任感
- **AIGC 方式**：需真人出镜，AI 辅助后期（背景替换、字幕、特效）
- **适合**：美妆、3C、保健品等需要专业背书的品类

### 产品展款（AIGC 最佳）
- **核心**：无口播，纯视觉锤，完播率高
- **结构**：产品特写 → 多角度展示 → 使用效果 → 品牌露出
- **AIGC 方式**：纯 AI 视频生成，最适合全流程 AIGC
- **适合**：服饰、珠宝、鞋包等视觉驱动品类

### 剧情演绎
- **核心**：脚本内容大于产品展示，情感驱动
- **AIGC 方式**：AI 生成场景 + 角色，适合品牌向内容
- **注意**：剧情要服务于产品，避免"看完不知道卖什么"

### 卖家促销
- **核心**：从工厂/仓库角度强调性价比和源头优势
- **AIGC 方式**：AI 生成工厂、仓库、流水线场景
- **适合**：白牌、工厂直销、性价比定位产品

### 明星测评（AIGC 慎用）
- **核心**：利用名人效应
- **AIGC 方式**：版权风险极高，不推荐用 AI 生成明星形象
- **建议**：如需此类型，使用真实授权素材

## 选择建议

```
客单价 < 50 元 → 产品展款 / 素人口播 / 卖家促销
客单价 50-200 元 → 情景种草 / 素人口播
客单价 > 200 元 → 达人测评 / 情景种草 / 剧情演绎
品宣为主 → 剧情演绎
视觉驱动品类 → 产品展款
```
