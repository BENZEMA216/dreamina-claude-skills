# 字幕样式规范

为千川素材设计字幕样式，提升可读性和转化效果。

## 字幕设计原则

### 1. 可读性优先
- 字体大小要够大，手机竖屏观看也能清晰
- 对比度要足够，确保在各种背景下都能看清
- 避免字幕被产品或重要画面遮挡

### 2. 强调重点
- 关键卖点用不同颜色或大小强调
- 价格、折扣等用醒目样式
- 品牌名可用特殊处理

### 3. 节奏配合
- 字幕出现时机与配音同步
- 每屏字数不要太多（建议 10-15 字）
- 给用户足够的阅读时间

## 字幕样式模板

### 标准卖货字幕
```
字体：思源黑体 / 阿里巴巴普惠体 / 抖音美好体
大小：48-56px（基于 1080x1920）
颜色：#FFFFFF（白色）
描边：#000000（黑色），3-4px
阴影：可选，轻微黑色阴影增加层次
位置：画面下方 1/4 处居中
动效：无或轻微弹出
```

### 强调型字幕（用于关键卖点）
```
字体：同上，加粗
大小：60-72px
颜色：#FFDD00（金黄色）或 #FF4D4F（红色）
描边：#000000，4px
位置：画面中央或略偏下
动效：放大弹出 + 轻微抖动
```

### 价格/优惠字幕
```
字体：同上，加粗
大小：72-88px
颜色：#FF4D4F（红色）或 #FF6B00（橙色）
描边：#FFFFFF（白色），4px
背景：可加半透明红色/橙色背景框
位置：画面中央醒目位置
动效：放大弹入
```

## 字体推荐

### 免费可商用
| 字体名 | 风格 | 适用场景 |
|--------|------|----------|
| 思源黑体 | 简洁现代 | 通用 |
| 阿里巴巴普惠体 | 现代商业 | 电商、卖货 |
| 抖音美好体 | 活泼友好 | 抖音原生风格 |
| 站酷系列 | 多种风格 | 按需选择 |
| OPPO Sans | 简洁大气 | 科技、数码 |

### 剪映内置
- 直接使用剪映内置字体，无版权风险
- 推荐：默认黑体、综艺体、卡通体

## 字幕动效建议

### 推荐动效
| 动效 | 适用场景 | 剪映设置 |
|------|----------|----------|
| 无动效 | 常规字幕 | 默认 |
| 淡入淡出 | 平稳过渡 | 入场淡入、出场淡出 |
| 逐字显示 | 配合配音节奏 | 打字机效果 |
| 弹出 | 强调重点 | 弹性入场 |

### 避免的动效
- 过于花哨的旋转、翻转
- 持续时间过长的动效
- 影响阅读的动效

## 按脚本类型的字幕建议

### 情景种草
- 风格：简洁、不抢画面
- 重点强调：卖点词、效果词
- 位置：下方 1/4

### 素人口播
- 风格：配合口播同步出字
- 重点强调：关键数据、效果
- 位置：下方，避开人脸

### 产品展款
- 风格：极简或无字幕
- 重点：产品名、价格
- 位置：底部或角落

### 卖家促销
- 风格：醒目、促销感
- 重点强调：价格、优惠、限时
- 位置：中央醒目位置

## 注意事项

1. **安全区域**：字幕不要太靠边，留出 5% 边距
2. **遮挡检查**：确认字幕不遮挡产品和人脸
3. **时长把控**：每条字幕至少显示 2 秒，让用户能读完
4. **一致性**：同一条视频内字幕风格保持统一
5. **测试检查**：在手机上预览，确保小屏也能看清
