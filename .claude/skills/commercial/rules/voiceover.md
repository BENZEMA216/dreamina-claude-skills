# AI 配音方案

为千川素材生成 AI 配音方案，包括配音文案处理和工具选择。

## 配音文案处理

### 格式标注
在文案中加入语气和停顿标记，便于 AI 配音工具识别：

```
【配音文案】
（语气：惊讶）哎？/ 你还在为 xxx 烦恼吗？//
（语气：平和）其实只需要这一招 /
（语气：强调）三天见效，七天蜕变 //
（语气：急促）现在下单还有优惠 / 点击下方购物车 /
```

### 标记说明
| 标记 | 含义 | 用途 |
|------|------|------|
| / | 短停顿 | 0.3-0.5秒，句内换气 |
| // | 长停顿 | 0.8-1秒，段落切换 |
| （语气：xxx） | 情感标注 | 告知 AI 语气变化 |
| 【重音】xxx | 重读标注 | 强调关键词 |

### 文案节奏原则
- 千川素材语速偏快，但不要快到听不清
- 推荐语速：每秒 4-5 个字
- 15秒素材：约 60-75 字
- 30秒素材：约 120-150 字

## 国内 AI 配音工具

### 豆包语音（推荐）
- **入口**：豆包 App / 火山引擎 TTS
- **优势**：自然度高，情感丰富，与抖音生态整合
- **推荐音色**：
  - 女声卖货：甜美女声、活力少女、知性女声
  - 男声卖货：阳光男声、磁性男声
- **语速**：建议 1.1-1.2x

### 通义语音
- **入口**：阿里云 TTS / 通义千问
- **优势**：稳定，批量处理能力强
- **推荐音色**：
  - 女声：小云、小燕
  - 男声：小刚
- **语速**：建议 1.0-1.1x

### 剪映 AI 配音
- **优势**：与剪映无缝集成，一站式完成
- **推荐场景**：后期已在剪映中完成的项目
- **音色**：剪映内置多种卖货向音色

## 海外 AI 配音工具

### ElevenLabs（首选）
- **优势**：最自然的 AI 语音，支持情感控制
- **推荐设置**：
  - Stability: 50-60%（太高会机械，太低会不稳定）
  - Similarity: 75-85%
- **推荐 Voice ID**：
  - 女声：Rachel, Bella, Elli
  - 男声：Adam, Josh, Antoni
- **特色**：支持克隆自定义音色

### PlayHT
- **优势**：性价比高，批量生成友好
- **推荐场景**：需要大量配音素材的矩阵投放

### Azure TTS
- **优势**：微软出品，多语言支持好
- **推荐场景**：海外市场投放、多语言版本

## 输出模板

```
【配音文案】
[带标记的完整口播文案]

【AI 配音工具推荐】
首选：[工具名]
- 音色：[具体音色名称]
- 语速：[推荐倍速]
- 情感/风格：[设置建议]

备选：[备选工具]
- 音色：[音色名称]

【文案时长估算】
总字数：xxx 字
推荐语速：xxx 字/秒
预计时长：约 xx 秒

【配音注意事项】
- [特殊处理建议，如某段需要加速/减速]
```

## 脚本类型配音建议

| 脚本类型 | 推荐音色风格 | 语气特点 |
|----------|--------------|----------|
| 情景种草 | 亲和、自然 | 像朋友推荐 |
| 素人口播 | 真实、平实 | 不要太播音腔 |
| 卖家促销 | 热情、急促 | 适度紧迫感 |
| 产品展款 | 无配音或轻柔旁白 | 让画面说话 |
| 达人测评 | 专业、可信 | 测评感 |

## 数字人 + 配音联动

如果使用数字人，配音需配合口型同步：
1. 先生成配音音频
2. 用音频驱动数字人口型
3. 推荐工具组合：
   - 豆包语音 + D-ID
   - ElevenLabs + HeyGen
   - 剪映 AI 配音 + 剪映数字人
