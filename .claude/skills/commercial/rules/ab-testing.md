# A/B 测试变体与多平台适配

完成主方案后，帮助用户生成测试变体和多平台适配版本。

## A/B 测试变体

### 为什么要做 A/B 测试
- 前 3 秒决定 90% 的效果，但最佳钩子无法预测
- 同一产品不同角度可能效果差异巨大
- 快速测试、数据说话、迭代优化

### 变体维度

#### 1. 前 3 秒钩子变体（最重要）
为每条素材准备 2-3 个不同的开头：

| 钩子类型 | 示例 | 适用场景 |
|----------|------|----------|
| 痛点提问 | "你是不是也有这个烦恼？" | 解决问题型产品 |
| 惊艳结果 | 直接展示使用后效果 | 效果可视化产品 |
| 数据冲击 | "100万人都在用" | 有数据背书的产品 |
| 身份圈定 | "所有宝妈注意了" | 精准人群产品 |
| 悬念引入 | "这个方法我用了3年" | 故事型种草 |
| 利益前置 | "今天只要9.9" | 促销型素材 |

#### 2. 卖点顺序变体
- 变体A：卖点1 → 卖点2 → 卖点3
- 变体B：卖点2 → 卖点1 → 卖点3
- 测试哪个卖点放在前面效果更好

#### 3. 结尾引导变体
- 变体A："点击下方购物车立即下单"
- 变体B："进入直播间还有更大优惠"
- 变体C："限时特价，点击抢购"

#### 4. 视觉风格变体
- 变体A：明亮温暖色调
- 变体B：高级冷淡色调
- 变体C：强对比撞色

### 变体生成输出格式

```
【A/B 测试变体方案】

原版（A版）：
- 前3秒钩子：[原版钩子描述]
- 画面 Prompt：[原版 prompt]

变体B：前3秒替换为痛点提问
- 钩子文案：[新文案]
- 画面 Prompt：[对应的新 prompt]
- 预期效果：吸引有明确需求的精准用户

变体C：前3秒替换为惊艳结果
- 钩子文案：[新文案]
- 画面 Prompt：[对应的新 prompt]
- 预期效果：用效果直接吸引眼球

【测试建议】
- 投放预算：每个变体至少 xxx 元测试
- 测试周期：xxx 天
- 关注指标：3秒完播率、整体完播率、点击率、转化率
```

## 多平台适配

### 画面比例适配

| 平台 | 比例 | 分辨率 | 适配方式 |
|------|------|--------|----------|
| 抖音/千川 | 9:16 | 1080x1920 | 主版本 |
| 快手 | 9:16 | 1080x1920 | 直接复用 |
| 小红书 | 3:4 | 1080x1440 | 裁切/重构图 |
| 微信视频号 | 9:16 | 1080x1920 | 直接复用 |
| 淘宝/天猫 | 1:1 | 1080x1080 | 裁切/重构图 |
| B站 | 16:9 | 1920x1080 | 横版重制 |

### 不同比例的 Prompt 适配

#### 9:16 → 1:1（方版）
- Midjourney：`--ar 1:1`
- 可灵/即梦：选择 1:1 比例
- 构图调整：主体居中，减少纵向空间

#### 9:16 → 16:9（横版）
- Midjourney：`--ar 16:9`
- 构图调整：增加横向环境元素，主体可偏左或偏右
- 适用：B站、横屏信息流

### 平台风格适配

| 平台 | 内容风格 | 适配建议 |
|------|----------|----------|
| 抖音 | 快节奏、强钩子 | 前3秒极致吸引 |
| 快手 | 真实、接地气 | 减少过度包装感 |
| 小红书 | 精致、种草感 | 增加生活方式元素 |
| 视频号 | 信息密度高 | 可适当加长 |

### 多平台输出格式

```
【多平台适配方案】

主版本（抖音/千川 9:16）：
[已完成的主方案]

小红书适配（3:4）：
- 画面裁切/重构建议：[描述]
- Prompt 调整：[新的 prompt，比例改为 3:4]
- 文案调整：[更偏种草风格的文案]
- 字幕位置调整：[新位置]

淘宝/天猫适配（1:1）：
- 画面裁切/重构建议：[描述]
- Prompt 调整：[新的 prompt，比例改为 1:1]
- 构图重点：产品居中，信息紧凑

横版适配（16:9）：
- 适用平台：B站、PC端
- Prompt 调整：[新的 prompt，比例改为 16:9]
- 构图重点：增加环境元素
```

## 使用流程

完成主方案后，询问用户：

1. **是否需要 A/B 测试变体？**
   - 如需要，生成 2-3 个前 3 秒钩子变体
   - 可选生成卖点顺序变体、结尾变体

2. **是否需要多平台适配？**
   - 如需要，询问目标平台
   - 生成对应比例和风格的适配方案

3. **是否需要换一种脚本类型？**
   - 同一产品可用不同脚本类型制作
   - 如情景种草版 + 产品展款版
