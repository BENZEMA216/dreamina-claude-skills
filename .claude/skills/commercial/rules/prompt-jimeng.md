# 即梦（Dreamina）Prompt 生成规则

为千川素材生成即梦 AI 的图片 Prompt。即梦使用中文 prompt，由字节跳动开发，与抖音生态深度整合。

## Prompt 结构

```
[主体描述]，[场景环境]，[光影氛围]，[色调/风格]，[构图方式]
```

## 即梦特色功能

### 风格选择
即梦提供预设风格，在 prompt 之外需选择：
- **写实摄影**：商业产品图、生活场景（千川素材首选）
- **3D 渲染**：产品三维展示
- **插画风格**：品牌向内容
- **动漫风格**：年轻受众群体

### 图片比例
- 9:16 竖屏（千川投放首选）
- 16:9 横屏
- 1:1 方图
- 3:4 竖图

### 参考图功能
- 支持上传参考图控制构图和风格
- 建议：先找到满意的竞品素材截图作为参考图，再用 prompt 替换产品和场景

## Prompt 撰写要求

### 主体描述
- 产品名称 + 外观细节（颜色、材质、形状）
- 人物描述用具体特征，避免抽象描述

### 场景描述
- 贴合目标人群生活场景
- 常用：现代客厅、简约浴室、明亮厨房、温馨卧室、户外花园、办公桌

### 光影与色调
- 千川素材建议：明亮、温暖、干净
- 避免：过暗、过冷、过于艺术化

### 构图
- 标注拍摄角度：正面、侧面45度、俯拍、仰拍
- 标注景别：特写、中景、全景

## 输出模板

```
--- 镜号 X：[画面描述摘要] ---

【即梦 Prompt】
[中文 prompt]
风格：写实摄影
画面比例：9:16
参考图：[是否建议使用参考图，如有建议描述参考图内容]
```

## 千川素材常用 Prompt 模式

### 产品特写
```
[产品名称]放在[表面材质]上，[颜色/材质细节]，柔和的影棚灯光，商业摄影风格，背景干净简洁，产品居中特写
风格：写实摄影
```

### 生活场景
```
一位[年龄段][性别]在[具体场景]中使用[产品名]，面带[表情]，[动作描述]，自然柔和的光线，温暖的色调，中景构图
风格：写实摄影
```

### 产品氛围图
```
[产品名]摆放在[有氛围感的场景]中，周围有[装饰元素]，[光影描述]，[色调]，[构图角度]，质感细腻
风格：写实摄影
```

### 对比图
```
左右对比构图，左侧展示[使用前状态]，右侧展示[使用后效果]，中间有清晰分隔线，明亮均匀的光线，干净背景
风格：写实摄影
```

## 即梦 vs 可灵选择建议

| 场景 | 推荐工具 | 原因 |
|------|----------|------|
| 抖音投放素材 | 即梦优先 | 与抖音生态整合，审核友好 |
| 需要图生视频 | 可灵优先 | 可灵视频生成能力更强 |
| 需要参考图控制 | 即梦优先 | 参考图功能更成熟 |
| 复杂运动场景 | 可灵优先 | 首帧-尾帧控制更精确 |
