# 创作手法选择

根据团队能力、资源和品类特点，按以下优先级选择创作手法：

## 1. 借鉴性模仿（首选，性价比最高）

- 参考竞品爆量素材结构，用 AIGC 重新生成画面
- 相比原创，爆量速度快、成本低
- 适合有大量竞品素材可参考的类目
- **核心原则：非必要，不原创**

工作方式：
1. 找到竞品爆量素材（用户提供或搜索）
2. 拆解其结构：开头钩子、卖点呈现方式、结尾引导
3. 保留结构，用 AI 重新生成全部画面
4. 替换为自己的产品卖点和文案

## 2. AIGC 原创

- 用 AI 生成全新画面和场景
- 适合竞品素材较少的蓝海品类
- 内容新鲜度高，但需要更多创意试错

工作方式：
1. 基于产品卖点构思原创脚本
2. 用 AI 工具生成全部视觉素材
3. 多做 A/B 测试，快速迭代

## 3. 混剪 + AI 增强

- 在已有素材基础上用 AI 替换背景、增强画质、生成新片段
- 制作成本最低
- 适合已有一定素材库的团队
- 注意：面临平台查重和同质化风险，需确保 AI 增强后差异度足够

## 选择决策树

```
有竞品爆量素材参考？
├── 是 → 借鉴性模仿
└── 否 → 有现成素材库？
    ├── 是 → 混剪 + AI 增强
    └── 否 → AIGC 原创
```
