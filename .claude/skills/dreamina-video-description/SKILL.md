# 视频描述规范

## 必须明确的要素
- **镜头运动**：推、拉、移、环绕、平移、跟随、手持
- **光线氛围**：明暗、虚化、光影变化
- **主体动作**：奔跑、转头、跳跃、咆哮（避免静态描写）
- **景别与视角**：远景、特写、航拍、仰拍
- **镜头切换**：保持场景与风格连贯性

## 语言要求
- 短句、简单词汇
- 突出特征："老人"、"戴墨镜的女人"
- 程度副词强化动作："快速"、"幅度大"、"疯狂"

## 示例
- `镜头快速前推到小女孩近景，她背对镜头缓缓抬头仰视面前高耸的建筑`
- `镜头迅速拉远，露出戴墨镜的女子上半身，她微微转头，目光快速移向画面右侧，背景是车流涌动的繁华街头`
- `高空航拍俯瞰沙漠，镜头环绕移动，展现一支骆驼商队缓慢前行`
